## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+ 
- npm, yarn, or pnpm

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd Portfolio
```

2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
```

3. Create a `.env.local` file (optional, for environment variables):
```bash
cp .env.example .env.local
```

4. Run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## ğŸ¨ Customization

### Colors

Edit `tailwind.config.ts` to customize the color scheme:
- `primary-bg`: Main background color
- `accent-rag`: Primary accent (cyan)
- `accent-code`: Secondary accent (purple)
- `accent-cloud`: Tertiary accent (blue)

### Content

Update component files in `src/components/` to customize:
- Personal information
- Projects data
- Skills and experience
- Social links
